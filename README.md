# 🛒 Flutter Ecommerce App

A modern **Ecommerce mobile application** built with Flutter using **Clean Architecture**, **BLoC state management**, and **Dio for API integration**.

---

## 📱 Features

### 👤 Customer App
- 🔐 Login & Register
- 🛍️ Products Listing
- 📄 Product Details
- 🛒 Cart Management
- 💳 Checkout Process
- 📦 Orders History

### 🧑‍💻 Admin App
- 📊 Dashboard
- ➕ Products CRUD (Create / Read / Update / Delete)
- 📦 Orders Management
- 👥 Users Management
